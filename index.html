Rem. REST easy.

Rem is a library for accessing any REST API simply. Whether using a third-party service or your own API, Rem only needs a few lines of code to move from reading API docs to making requests. And with support for popular services and authentication built-in, you might never again need a dedicated API library.


var rem = require('rem');
var gh = rem('github.com');

rem.console(gh, {
  scope: ["user", "repo"]
}, function (err, user) {
  user('user').get(function (err, profile) {
    console.log(profile);
  });
});
npm install rem or Download rem.js or View on Github
